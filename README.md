# ScriptWeaver

ScriptWeaver is an AI-assisted tool for helping novel authors convert multi-chapter novel text into editable screenplay drafts.

## Development

Create and activate a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Install the project with development dependencies:

```bash
python3 -m pip install -e ".[dev]"
```

Run tests:

```bash
pytest
```

## Contributing

Each pull request should implement one focused feature and keep the main branch runnable after merge.
